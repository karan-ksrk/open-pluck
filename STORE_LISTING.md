## Detailed Description

```
Open Pluck lets you highlight any text in a ChatGPT or Claude.ai conversation and ask follow-up questions about it — without cluttering your main chat, breaking your flow, or wasting context window tokens.

HOW IT WORKS
1. Open ChatGPT or Claude.ai and have any conversation
2. Highlight any text inside an AI response
3. A popover appears with quick options — "What is this?", "Explain this", or type a custom question
4. A popup window opens and auto-submits your question using your logged-in session
5. A sidebar tracks all your questions for the session

Your main conversation stays completely untouched — no extra messages, no context pollution, no token waste.

FEATURES
• Completely free — no API keys, no subscriptions, no configuration
• One-click questions — "What is X?", "Explain this", or custom input
• Popup window — answer opens in a dedicated window alongside your chat
• Session sidebar — tracks all your highlight questions in one place
• Private — no data collected, no external servers, uses your own browser session
• Non-intrusive — works on text inside AI responses only

SUPPORTED SITES
• ChatGPT (chatgpt.com, chat.openai.com)
• Claude.ai (claude.ai)

Requirement: You must be logged in to your chosen service (ChatGPT or Claude.ai) in Chrome for the extension to work.
```

## Privacy Certification Fields (Dashboard)

When filling out the Chrome Web Store privacy form:

| Field | Answer |
|-------|--------|
| **Single purpose** | Lets users highlight text in ChatGPT/Claude AI responses and ask follow-up questions in a separate popup window. |
| **Permission justification** | storage: save service preference and pass question to popup; windows/tabs: open popup window; host permissions: inject UI and submit questions on these sites only. |
| **Remote code** | No remote code used. |
| **Data collection** | No data collected. Uses chrome.storage.local only for preferences and temporary question passing. |
